# Mage-Defender
Made Defender project in Unity

Key features implemented in this game:
  -Tower defence game mechanics (Enemies with a defined path, Towers, Upgrades, etc)
  -3D gameplay with camera movement and rotation
  -save system
  -shop system (Towers purchasing and upgrading)

The game is fnished and can be found on my itch.io page: https://mariusdev.itch.io/mage-defender
